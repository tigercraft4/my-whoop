# Plan 09-01: DesignTokens + RootTabView Redesign

## Objective

Actualizar `DesignTokens.swift` com os tokens de cor WHOOP confirmados via `docs/whoop-ui-reference.md` e redesenhar `RootTabView` com 5 tabs em ordem correcta (Today / Sleep / Strain / Trends / Device) + `@SceneStorage` para persistência de tab.

## Context

`DesignTokens.swift` já tem `recoveryGreen`, `recoveryYellow`, `recoveryRed`, `strainBlue` definidos. A tarefa é auditar esses valores contra `docs/whoop-ui-reference.md` e confirmar/corrigir + adicionar `WH.Color.strainAccent` se não existir com este nome. `RootTabView` actualmente tem 5 tabs mas sem `@SceneStorage` e com ordem incorrecta (Trends antes de Workouts, sem Strain tab dedicada).

   - `recoveryGreen = #16EC06` — confirmar vs referência WHOOP
   - `recoveryYellow = #FFDE00` — confirmar vs referência WHOOP

<task id="09-01-T2">
<type>execute</type>
<title>Redesenhar RootTabView com 5 tabs correctas + @SceneStorage</title>

<read_first>
- `ios/OpenWhoop/App/RootTabView.swift` — implementação actual (4 tabs, sem SceneStorage)
- `ios/OpenWhoop/Tabs/WorkoutsView.swift` — será substituída/wrappada por StrainView
- `.planning/phases/09-swiftui-redesign-whoop-style/09-CONTEXT.md` — D-01, D-02, D-03
- `docs/whoop-ui-reference.md` — ícones e labels confirmados para cada tab
</read_first>

<action>
1. Em `RootTabView.swift`, adicionar `@SceneStorage("selectedTab") private var selectedTab = "today"` como property.
2. Mudar `TabView { ... }` para `TabView(selection: $selectedTab) { ... }`.
3. Definir as 5 tabs em ordem com tags string para o `@SceneStorage` binding:
   - Tab 1: `TodayView()` — tag `"today"`, Label("Today", systemImage: "house")
   - Tab 2: `SleepView()` — tag `"sleep"`, Label("Sleep", systemImage: "bed.double")
   - Tab 3: `StrainView()` — tag `"strain"`, Label("Strain", systemImage: "bolt.heart") — usar `WorkoutsView()` temporariamente se `StrainView` não existir ainda no Wave 3
   - Tab 4: `TrendsView()` — tag `"trends"`, Label("Trends", systemImage: "chart.xyaxis.line")
   - Tab 5: `NavigationStack { LiveView() }` — tag `"device"`, Label("Device", systemImage: "wave.3.right")
4. Adicionar `.tag("today")`, `.tag("sleep")`, etc. a cada tab child usando `.tabItem { ... }.tag("today")`.
5. Manter `.preferredColorScheme(.dark)` no TabView se existia.
6. Nota: se `StrainView` ainda não existir (será criada no Plan 09-03), usar `WorkoutsView()` com tag `"strain"` como placeholder — o Plan 09-03 criará `StrainView` e actualizará esta referência.
</action>

<acceptance_criteria>
- `RootTabView.swift` contém `@SceneStorage("selectedTab")` property
- `TabView(selection: $selectedTab)` é usado (não `TabView { }` sem selection)
- 5 tabs presentes em ordem: Today / Sleep / Strain (ou WorkoutsView placeholder) / Trends / Device
- Cada tab tem `.tag("today")`, `.tag("sleep")`, `.tag("strain")`, `.tag("trends")`, `.tag("device")`
- Projecto compila sem erros
- Seleccionar uma tab e reiniciar a app mantém a tab seleccionada (verificar com `@SceneStorage` a preservar o estado)
</acceptance_criteria>
