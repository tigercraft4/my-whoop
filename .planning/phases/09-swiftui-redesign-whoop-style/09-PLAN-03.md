# Plan 09-03: RecoveryCard — TodayView WHOOP-Style

## Objective

Criar `RecoveryCard.swift` com anel de cor WHOOP-style (usando `ZoneRingView`) e integrar na `TodayView` como hero section. O card mostra recovery score 0–100 com cor por zona, HRV RMSSD, RHR, e sleep performance — todos de `DailyMetric`. Placeholder `"—"` quando dados são `nil`.

## Context

`TodayView` actualmente tem uma `heroSection` com lógica de recovery. A tarefa é extrair essa lógica para `RecoveryCard.swift` e actualizá-la para usar `ZoneRingView` com cores WHOOP. O `DailyMetric` já tem `recovery` (Double? 0–1 fraction), `avgHrv` (Double?), `restingHr` (Int?), e `efficiency` (Double? 0–1). O card usa `.background(Color.black)` conforme D-12.

<action>
1. Criar `ios/OpenWhoop/Design/Components/RecoveryCard.swift`.
2. Implementar `RecoveryCard` como `View` que recebe `daily: DailyMetric?` como parâmetro.
3. Layout do card (background preto conforme D-12):
   - `.background(Color.black).cornerRadius(WH.Radius.card)`
   - Padding interno: `WH.Spacing.lg`
4. Conteúdo principal:
   - Header: `Text("RECOVERY").font(WH.Font.cardTitle).foregroundStyle(WH.Color.textSecondary).kerning(1.5)`
   - Hero ring: `ZoneRingView` centrado com:
     - `value`: `(daily?.recovery ?? 0) * 100` (converter de 0-1 para 0-100)
     - `maxValue`: `100`
     - `color`: `WH.Color.recoveryColor(forPercent: (daily?.recovery ?? 0) * 100)` — se `daily?.recovery == nil`, usar `.recoveryRed` como fallback
     - `size: 160`
     - `lineWidth: 20`
     - `centerLabel`: score formatado como inteiro (ex: "73") ou `"—"` se nil
   - Score label grande abaixo do anel: `Text(scoreText).font(WH.Font.metricHero()).foregroundStyle(Color.white)` — onde `scoreText` é `"—"` se `daily?.recovery == nil`, senão `"\(Int((daily!.recovery! * 100).rounded()))"`
     - Nota: se o centerLabel já mostra o score dentro do anel, este label abaixo pode mostrar o label RECOVERY ou omitir; decidir visualmente
   - Stats row horizontal (3 colunas):
     - HRV: label `"HRV"` + valor `daily?.avgHrv.map { "\(Int($0)) ms" } ?? "—"`
     - RHR: label `"RHR"` + valor `daily?.restingHr.map { "\($0) bpm" } ?? "—"`
     - SLEEP: label `"SLEEP"` + valor `daily?.efficiency.map { "\(Int($0 * 100))%" } ?? "—"`
   - Cada stat: `VStack(spacing: 2) { Text(label).font(WH.Font.cardTitle); Text(value).font(WH.Font.metricMedium()) }`
5. Placeholder state: se `daily == nil`, mostrar anel cinzento (ringTrack color) com `"—"` no centro + todos os stats `"—"`
6. Adicionar `#Preview` com um `DailyMetric` de exemplo (recovery: 0.73, avgHrv: 52.0, restingHr: 56, efficiency: 0.84)
</action>
