# Plan 09-05: StrainCard + StrainView — Tab Strain WHOOP-Style

## Objective

Criar `StrainCard.swift` com anel 0–21 (usando `ZoneRingView` parametrizado com `WH.Color.strainAccent`) e criar `StrainView.swift` que mostra `StrainCard` no topo + lista de workouts abaixo. Actualizar `RootTabView` para usar `StrainView` em vez de `WorkoutsView` no tab "Strain".

## Context

O CONTEXT.md D-08 e D-09 definem: StrainView = StrainCard (topo) + lista de workouts (baixo). `WorkoutsView` já tem a lógica de lista de workouts — será reutilizada dentro de `StrainView`. O anel do StrainCard usa o mesmo `ZoneRingView` do RecoveryCard mas parametrizado para range 0–21 e cor `strainAccent` fixa.
