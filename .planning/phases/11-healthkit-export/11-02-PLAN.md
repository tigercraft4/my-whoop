<objective>
Create the `HealthKitExporter` actor that exports HR samples (HK-01), HRV RMSSD per sleep session (HK-02), and sleep sessions with stage mapping (HK-04) to Apple Health. Uses highwater cursors in UserDefaults for idempotent HR/HRV export and delete+reinsert strategy for sleep sessions. SpO₂ (HK-03) is entirely omitted — no dead code, no feature flags.
</objective>

<threat_model>
- MEDIUM: Delete+reinsert for sleep (`deleteObjects`) is irreversible if the session window predicate is too broad — could delete data from other Health apps (mitigated: predicate uses exact session start/end timestamps from `CachedSleepSession`).
- LOW: Highwater cursor persisted to UserDefaults; UserDefaults can be cleared by the OS in low-storage. In that case the export re-runs from epoch 0, producing duplicate HK writes — HealthKit deduplicates by `HKSource` + time window so duplicates are harmless for samples; sleep delete+reinsert is already idempotent.
- LOW: HealthKit authorization errors are not surfaced to the UI in this plan (surfaced in Plan 11-03). Exporter silently no-ops when not authorized.
</threat_model>

    <action>
      Create ios/OpenWhoop/HealthKit/HealthKitExporter.swift (create the HealthKit/ directory if needed).

      Implement as a Swift `actor HealthKitExporter`:

      **Properties:**
      - `private let store: HKHealthStore`
      - `private let whoopStore: WhoopStore`
      - `private let deviceId: String`
      - UserDefaults keys as static constants:
        - `static let hrHighwaterKey = "hk.hrHighwater"` (TimeInterval / epoch seconds)
        - `static let hrvHighwaterKey = "hk.hrvHighwater"` (TimeInterval / epoch seconds; not used for sleep — delete+reinsert handles idempotence)

      **`requestAuthorization() async throws`:**
      - Request write access (share) for: `HKQuantityType(.heartRate)`, `HKQuantityType(.heartRateVariabilitySDNN)`, `HKCategoryType(.sleepAnalysis)`
      - Returns normally if authorized; caller handles denial via `authorizationStatus(for:)` check
      - Does NOT request read access (toRead = empty set — HK-05 only requires write)

      **`export() async`:**
      - Calls `exportHR()`, `exportHRV()`, `exportSleep()` in sequence
      - Catches and logs errors per stream; one failure does not abort the others

      **`private func exportHR() async throws`:**
      - Read highwater from UserDefaults: `let since = Int(UserDefaults.standard.double(forKey: Self.hrHighwaterKey))`
      - Query `whoopStore.hrSamples(deviceId: deviceId, since: since, limit: 5000)`
      - Convert each HRSample to `HKQuantitySample`:
        - Type: `HKQuantityType(.heartRate)`
        - Quantity: `HKQuantity(unit: HKUnit(from: "count/min"), doubleValue: Double(sample.bpm))`
        - Start/end: `Date(timeIntervalSince1970: TimeInterval(sample.ts))` (point sample, start == end)
        - Metadata: `[HKMetadataKeyWasUserEntered: false]`
      - Save via `store.save(samples)`
      - Update highwater to `samples.last!.ts` on success

      **`private func exportHRV() async throws`:**
      - Read HRV highwater: `let since = Int(UserDefaults.standard.double(forKey: Self.hrvHighwaterKey))`
      - Query `whoopStore.sleepSessions(deviceId: deviceId)` (all sessions)
      - Filter: `sessions.filter { $0.endTs > since && $0.avgHrv != nil }`
      - For each session with `avgHrv`:
        - Type: `HKQuantityType(.heartRateVariabilitySDNN)`
        - Quantity: `HKQuantity(unit: .secondUnit(with: .milli), doubleValue: session.avgHrv!)`
        - Start: `Date(timeIntervalSince1970: TimeInterval(session.startTs))`
        - End: `Date(timeIntervalSince1970: TimeInterval(session.endTs))`
        - Metadata: `[HKMetadataKeyWasUserEntered: false]`
      - Save via `store.save(samples)`
      - Update HRV highwater to `max(sessions.map { $0.endTs })` on success

      **`private func exportSleep() async throws`:**
      - Query `whoopStore.sleepSessions(deviceId: deviceId)`
      - For each session:
        1. DELETE existing HK sleep samples for the exact session window:
           - Predicate: `NSPredicate(format: "startDate >= %@ AND endDate <= %@", sessionStart as NSDate, sessionEnd as NSDate)`
           - `try await store.deleteObjects(of: HKCategoryType(.sleepAnalysis), predicate: predicate)`
        2. Parse `session.stagesJSON` — JSON array of `{start: Int, end: Int, stage: String}` objects
        3. Map stage strings to HKCategoryValueSleepAnalysis:
           - "light" or "core" → `.asleepCore`
           - "deep" → `.asleepDeep`
           - "rem" → `.asleepREM`
           - "awake" → `.awake`
           - unknown → skip (log warning)
        4. Create one HKCategorySample per stage segment
        5. Save all samples for the session in a single `store.save(batch)`

      **HK-03 SpO₂:** Zero code, zero references. Not implemented, not gated, not commented as TODO.
