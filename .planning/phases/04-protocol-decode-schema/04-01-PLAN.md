Build `re/survey_5/decode_5.py` — the Maverick-aware body decoder that is the reference implementation for all of Phase 4 — and run the D-01 body-layout gate against the 46-frame golden corpus. Output: `re/survey_5/decode_5.py` (library, body decoder + dispatch), expanded `re/survey_5/frames_5_golden.json` (curated, with `stream_type`).
