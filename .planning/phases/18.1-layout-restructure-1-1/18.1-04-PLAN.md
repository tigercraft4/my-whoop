### Task 4.2 — Build gate completo final

<read_first>
- Todos os ficheiros modificados nesta fase (RootTabView.swift, HomeView.swift, HealthView.swift, CommunityView.swift, MoreView.swift)
</read_first>

<action>
Executar o build gate oficial da fase:

```bash
cd ios && xcodegen generate && xcodebuild build \
  -scheme OpenWhoop \
  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' \
  2>&1 | tee /tmp/build_18_1_final.log | grep -E "error:|warning:|BUILD SUCCEEDED|BUILD FAILED"
```

Se BUILD FAILED:
1. Examinar erros: `grep 'error:' /tmp/build_18_1_final.log | head -20`
2. Categorizar erros por ficheiro e corrigir na ordem:
   - Erros em `RootTabView.swift` → fixar referências a views
   - Erros em `HomeView.swift` → fixar tipos, environment objects
   - Erros em `HealthView.swift` → fixar TrendPoint/MetricKind
   - Erros em `CommunityView.swift` ou `MoreView.swift` → fixar referências
3. Re-executar build até BUILD SUCCEEDED

Avisos (warnings) são aceitáveis desde que não sejam erros.
</action>

<acceptance_criteria>
- `cd ios && xcodegen generate && xcodebuild build -scheme OpenWhoop -destination 'platform=iOS Simulator,name=iPhone 17 Pro' 2>&1 | tail -3` contém "BUILD SUCCEEDED"
- `grep '^.*error:' /tmp/build_18_1_final.log 2>/dev/null | wc -l` retorna 0
- Build completado em menos de 5 minutos (sem compilação excessiva de ficheiros não modificados)
</acceptance_criteria>
