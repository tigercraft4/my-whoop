## Summary

Substituídos os stubs de Wave 1 com implementações completas: `HealthView` com 4 secções de métricas (HRV, RHR, SpO₂, sono 7 dias via `MetricChart`); `CommunityView` definitiva com "Em breve"; `MoreView` com deviceId, app version e link para SettingsView.

## What was built

SpO₂ mostrado como último valor (sem gráfico). Usa `WH.Color.teal`, `WH.Color.sleepPurple`, `WH.Color.textPrimary`.
- **CommunityView.swift**: `Image(systemName: "person.3")` + "Em breve" + texto explicativo em pt-PT + toolbar "COMUNIDADE".
- **MoreView.swift**: `List` com sections DISPOSITIVO (AppConfig.deviceId), APP (Bundle version + NavigationLink para SettingsView), INFORMAÇÃO (WHOOP 5.0 BLE). Usa `WH.Color.background`, `WH.Color.textPrimary`, `WH.Color.textSecondary`.
