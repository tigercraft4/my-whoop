## Summary

`HomeView.swift` completamente reestruturado com header customizado de navegação de data, 3 `ZoneRingView` em linha (Sono/Recuperação/Esforço com tap para detail views), e 5 secções de scroll que replicam o layout do WHOOP 5.37.0.

## What was built

- **Header customizado** (`homeHeader`): esquerda (perfil + strain), centro (< HOJE / data >) com `@State selectedDate` e navegação por dia, direita (recovery% + indicador BLE).
- **3 Rings em linha** (`ringsSection`): `ZoneRingView(size: 90, lineWidth: 8)` para SONO (sleepPurple), RECUPERAÇÃO (recoveryColor dinâmico), ESFORÇO (strainBlue), cada um como `NavigationLink` para SleepView/MetricDetailView(.recovery)/StrainView.
- **Dados em falta card**: mostrado quando `effectiveMetric == nil && !metrics.isRefreshing && selectedDate.isToday`.
- **Monitor de Saúde + Stress** (2 cards lado a lado): HRV e RHR actuais, navega para HealthView.
- **O meu dia**: header de secção com botão [+] desactivado.
- **Atividades de Hoje**: 2 botões stub ("ADICIONAR ATIVIDADE" / "INICIAR ATIVIDADE") desactivados.
- **Sono desta noite**: mostra `sleepNeededMin` formatado + link para AlarmView.
- **Navegação de data**: `.task(id: selectedDate)` carrega dados via `metrics.daily(fromDay:toDay:)` para dias anteriores.
