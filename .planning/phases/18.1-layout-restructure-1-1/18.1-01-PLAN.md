# Plan 18.1-01 — LAYOUT-01: Nova Tab Bar + Botão W Overlay

## Objective

Substituir a tab bar de 5 tabs actuais (Today/Sleep/Strain/Trends/Device) por 4 tabs WHOOP-style (Início/Saúde/Comunidade/Mais) em `RootTabView.swift`, e mover o LiveView para um botão overlay circular "W" no canto inferior direito. Stubs (`CommunityView`, `MoreView`, `HealthView`) serão referenciados aqui e criados no plano 18.1-03.

## Context

A tab bar actual tem 5 tabs com nomes em inglês. O WHOOP 5.37.0 tem 4 tabs + botão W separado. As novas views `HomeView`, `HealthView`, `CommunityView`, `MoreView` serão criadas nos planos 18.1-02 e 18.1-03. Este plano cria `RootTabView.swift` com stubs de compilação onde necessário, e `HomeView.swift` como renomeação stub temporária para desbloquear a build enquanto o plano 18.1-02 o preenche completamente.

<action>
Substituir completamente `ios/OpenWhoop/App/RootTabView.swift`:

1. `@SceneStorage("selectedTab") private var selectedTab = "home"` — chave alterada de "today" para "home"
2. `@State private var showLive = false` — controla sheet/overlay para LiveView
3. `TabView(selection: $selectedTab)` com 4 tabs:
   - Tab "home" → `HomeView()` com `.tabItem { Label("Início", systemImage: "house") }`
   - Tab "health" → `HealthView()` com `.tabItem { Label("Saúde", systemImage: "heart.text.clipboard") }`
   - Tab "community" → `CommunityView()` com `.tabItem { Label("Comunidade", systemImage: "person.3") }`
   - Tab "more" → `MoreView()` com `.tabItem { Label("Mais", systemImage: "line.3.horizontal") }`
4. Envolver `TabView` num `ZStack(alignment: .bottomTrailing)` para o botão W overlay
5. Botão W overlay:
   - `Button { showLive = true } label: { ... }` com `.padding(.trailing, 20).padding(.bottom, 8)`
   - Label: `ZStack { Circle().fill(WH.Color.sleepPurple).frame(width: 52, height: 52); Text("W").font(.system(size: 22, weight: .bold)).foregroundStyle(.white) }`
   - `.shadow(color: WH.Color.sleepPurple.opacity(0.4), radius: 8, x: 0, y: 4)`
6. `.sheet(isPresented: $showLive) { NavigationStack { LiveView() } }`
7. Manter `.preferredColorScheme(.dark)` no ZStack/TabView
</action>
