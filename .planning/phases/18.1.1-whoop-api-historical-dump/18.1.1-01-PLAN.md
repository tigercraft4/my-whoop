# Plan 18.1.1-01 — CSV Export Import para iOS GRDB

## Objective

Script Python one-shot que lê o ZIP do export WHOOP (ciclos_fisiológicos.csv ou em inglês) e insere todos os registos históricos na BD GRDB do iOS, para ter ground truth local antes de o premium acabar.

## Context

A app WHOOP permite exportar todos os dados via Perfil → Privacidade → Exportar dados (CSV por email). O `import_csv_to_grdb.py` parseia o CSV em pt-PT ou en, normaliza os headers com unicode decomposition, e faz upsert via `INSERT OR REPLACE` na tabela `dailyMetric` do WhoopStore SQLite.

## Tasks

### Task 1.1 — Script `import_csv_to_grdb.py`

Criar `server/ingest/app/whoop_api/import_csv_to_grdb.py` com:
- Parser de headers pt-PT/en via `unicodedata.normalize` + diacritics stripping
- Mapeamento completo: recovery%, hrv(ms), restingHr, spo2%, skinTemp, respRate, sleepPerf%, sleepNeeded, totalSleep, deep, rem, light, awake → disturbances, strain
- `INSERT OR REPLACE INTO dailyMetric` com todos os campos
- `--zip`, `--db`, `--device-id` como argumentos CLI
- Dry-run output de quantas linhas inseridas

**Self-check:** `python import_csv_to_grdb.py --zip whoop.zip --db whoop.sqlite --device-id X` → sem erros, N rows inseridas.
