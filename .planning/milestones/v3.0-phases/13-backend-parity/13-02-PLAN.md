Assinatura exacta:
```
def sleep_performance_score(
    total_sleep_min: float,
    efficiency: float,
    deep_min: float,
    rem_min: float,
    disturbances: int,
    sleep_needed_min: float | None = None,
) -> float:
```

Fórmula composta (pesos ALG-10: W_dur=0.45, W_eff=0.25, W_stg=0.20, W_con=0.10):
- `target = sleep_needed_min if sleep_needed_min is not None else 420.0`
- `w_dur = min(total_sleep_min / max(target, 1.0), 1.0) * 0.45`
- `w_eff = efficiency * 0.25` (efficiency já é 0.0–1.0, TST/TIB)
- `restorative_ratio = (deep_min + rem_min) / max(total_sleep_min, 1.0)`; `w_stg = min(restorative_ratio / 0.40, 1.0) * 0.20`
- `w_con = (1.0 - min(disturbances / 10.0, 1.0)) * 0.10`
- `score = (w_dur + w_eff + w_stg + w_con) * 100`
- Retornar `round(max(0.0, min(100.0, score)), 1)`
