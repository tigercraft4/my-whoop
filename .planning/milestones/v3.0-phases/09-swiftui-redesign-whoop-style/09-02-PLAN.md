# Plan 09-02: ZoneRingView — Componente de Anel Partilhado

## Objective

Criar `ZoneRingView` — um componente SwiftUI de anel circular parametrizável que serve tanto o `RecoveryCard` (0–100, cor por zona) como o `StrainCard` (0–21, `strainAccent` fixo). Implementado como Canvas arc shape para controlo total sobre stroke width, linecap arredondado, e fundo de track.

## Context

O CONTEXT.md (Specific Ideas) sugere `ZoneRingView(value: Double, maxValue: Double, color: Color)`. O RecoveryCard precisa de cor dinâmica baseada em zona (via `WH.Color.recoveryColor(forPercent:)`), o StrainCard usa `WH.Color.strainAccent` fixo. O componente é a peça central do redesign WHOOP-style.

<action>
1. Criar `ios/OpenWhoop/Design/Components/ZoneRingView.swift`.
2. Implementar `ZoneRingView` como `View` com parâmetros:
   - `value: Double` — valor actual (ex: 73.0 para recovery, 14.2 para strain)
   - `maxValue: Double` — máximo do range (100.0 para recovery, 21.0 para strain)
   - `color: Color` — cor do arco (passada pelo chamador; para recovery é `WH.Color.recoveryColor(forPercent: value)`, para strain é `WH.Color.strainAccent`)
   - `lineWidth: CGFloat = 16` — espessura do arco (default razoável para cards)
   - `size: CGFloat = 120` — diâmetro do anel (ajustável por contexto)
3. Implementação com Canvas ou ZStack de Circles:
   - Track ring: `Circle().trim(from: 0, to: 1).stroke(WH.Color.ringTrack, style: StrokeStyle(lineWidth: lineWidth, lineCap: .round))`
   - Progress ring: `Circle().trim(from: 0, to: CGFloat(min(value / maxValue, 1.0))).stroke(color, style: StrokeStyle(lineWidth: lineWidth, lineCap: .round)).rotationEffect(.degrees(-90))`
   - Usar `ZStack` com `.frame(width: size, height: size)` como container
4. Adicionar label central opcional via ViewBuilder ou parâmetro de texto:
   - `centerLabel: String? = nil` — texto a mostrar no centro do anel (ex: "87" para recovery score)
   - Se `centerLabel != nil`, mostrar `Text(centerLabel!).font(WH.Font.metricLarge()).foregroundStyle(Color.white)` centrado
5. Adicionar preview com valores de exemplo:
   - Recovery ring: `ZoneRingView(value: 73, maxValue: 100, color: WH.Color.recoveryGreen, centerLabel: "73")`
   - Strain ring: `ZoneRingView(value: 14.2, maxValue: 21, color: WH.Color.strainAccent, centerLabel: "14.2")`
   - Red zone: `ZoneRingView(value: 25, maxValue: 100, color: WH.Color.recoveryRed, centerLabel: "25")`
</action>
